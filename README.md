
# QR Attendance Pro

## Setup
1. Configure .env
2. npm install (in backend and bot)
3. Run backend: npm run dev
4. Run bot: node bot.js

## Env example
DISCORD_CLIENT_ID=
DISCORD_CLIENT_SECRET=
DISCORD_CALLBACK=http://localhost:3000/auth/discord/callback
SESSION_SECRET=secret
BOT_TOKEN=
GUILD_ID=
PROFESSOR_ROLE=Professor
